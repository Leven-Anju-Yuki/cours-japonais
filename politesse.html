Formules de Politesse en Japonais

こんにちは (コンニチハ)

Romaji : Konnichiwa

Traduction : Bonjour

おはようございます (オハヨウゴザイマス)

Romaji : Ohayou gozaimasu

Traduction : Bonjour (le matin)

こんばんは (コンバンハ)

Romaji : Konbanwa

Traduction : Bonsoir

すみません (スミマセン)

Romaji : Sumimasen

Traduction : Excusez-moi / Pardon

さようなら (サヨウナラ)

Romaji : Sayōnara

Traduction : Au revoir

ありがとうございます (アリガトウゴザイマス)

Romaji : Arigatou gozaimasu

Traduction : Merci

どういたしまして (ドウイタシマシテ)

Romaji : Douitashimashite

Traduction : De rien

おやすみなさい (オヤスミナサイ)

Romaji : Oyasumi nasai

Traduction : Bonne nuit

いらっしゃいませ (イラッシャイマセ)

Romaji : Irasshaimase

Traduction : Bienvenue (dans un magasin)

お疲れ様です (オツカレサマデス)

Romaji : Otsukaresama desu

Traduction : Merci pour votre travail
